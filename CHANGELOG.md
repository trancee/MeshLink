# Changelog

This project keeps a release-oriented changelog.

## 0.1.0 (2026-07-15)


### ⚠ BREAKING CHANGES

* **meshlink:** MeshLink.updateBattery() has been removed. Battery-driven automatic power policy now requires no host app integration.
* **crypto:** cryptographically bind mesh domain into Noise XX handshake
* **engine:** remove self-asserted sender key fields from DirectMessageEnvelope
* **engine:** pin end-to-end trust only from authenticated Noise handshakes
* **wire:** add relayed end-to-end handshake wire frames and relay dispatch
* **api:** replace BleTransportBridge Any-typed callback with an enable/disable flag
